## KI Prototyp
Dieses Projekt wurde für das Modul **4FSC0PD004** am **SAE Institute Hannover** erstellt.

Entwicklungs-Zeitraum: **21.04.2025 - 02.05.2025**  

Aufgaben:
- Erstellung einer Menge von NPCs (Non-Player-Characters)
- Implementierung von mindestens 2 verschiedene Verhaltensmuster:
  - Chase & Evade (verfolgen & ausweichen)
  - Idle & Attack (Untätigkeit & Angriff)

Features:
- Unity Nav Mesh System
- Zombies-Vehalten mit FSM-Struktur (Finite State Machine)
- Charakters Bewegung mit Input Action

------------------------------------------------------------------------

## Simulation der Spielumgebung
Dieses Projekt wurde für das Modul **4FSC0PD004** am **SAE Institute Hannover** erstellt.

Entwicklungs-Zeitraum: **09.06.2025 - 13.06.2025**  

Aufgaben:
- Simulation natürlicher Oberflächen und Effekte im Zusammenspiel mit den Spielregeln einer Welt
- Simulation mind. 3 verschiedener Oberflächen und Objekte wie z.B.
  - Wasser
  - Sand
  - Metall
  - Gras
- Simulation mind. 3 verschiedener visuellen Effekten wie z.B.
  - Rauch
  - Regen
  - Nebel
  - Feuer
 
Features:
- Erstellung von Terrain mit Procedural-Level-Tool
- Feuer/Rauch/Regen-Simulation mit Particle System und derren Zusammenhang-Logik

## Threadoptimierung
Dieses Projekt wurde für das Modul **4FSC0PD003** am **SAE Institute Hannover** erstellt.

Entwicklungs-Zeitraum: **14.07.2025 - 25.07.2025**
- Update am 11.09.2025

Aufgaben:
- Nutzung von Multithreading-Techniken
- Erreichen der Performanceoptimierung von circa 10% der Ausgangsperformance

Features:
- Kombination von Unitys Jobsystem mit KI-Prototyps FSM
- Erfassung von Daten mit Profiler und Frame-Rate-Counter

------------------------------------------------------------------------

## Prozedurale Erweiterung der Spielwelt
Dieses Projekt wurde für das Modul **4FSC0PD004** am **SAE Institute Hannover** erstellt.

Entwicklungs-Zeitraum: **26.07.2025 - 01.08.2025**

Aufgaben:
- Entwicklung von Scripts und UI-Elemente für die Level-Generierung im Editor
- Vorgeschlagene Features:
  - Dungeon/Innen-Level
    - Größe des Levels (evtl. unendlich groß)
    - Unterschiedliche Level-Typen (Räume, Gänge, Labyrinthe,...)
    - Platzierungsmöglichkeiten für NPCs, Items,...
    - Verschiedene Grafiksets
  - Landschaft/Terrain
    - Größe des Levels (evtl. unendlich groß)
    - Beschaffenheit der Landschaft (hügelig, flach, bergig,...)
    - Unterschiedliche Biome (Wüste, Wald, Tundra, Graslandschaft,...)
    - Texturierung der der Landschaft / bzw. der Biome
    - Plateaus auf verschiedenen Ebenen
    - Platzierungsmöglichkeiten für Gebäude, Pflanzen,...

Features:
- Dungeon-Generator mit Einstellungen von Größe, Typ, Grafikset, Spawnmöglichkeiten für NPCs/Items
- Terrain-Generator mit Einstellungen von Größe, Biome, Höhenvariation, Platzierung von Objekten
- Editor Window
