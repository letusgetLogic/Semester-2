## KI Prototyp
Dieses Projekt wurde für das Modul **4FSC0PD004** am **SAE Institute Hannover** erstellt.

Entwicklungs-Zeitraum: **04.11.2024 - 28.02.2025**  
- Update am 06.01.2026

Aufgaben:
- Erstellung eines Spiel unter Verwendung eines Frameworks oder einer Game-Engine
- Implementierung von grundlegende Spielmechaniken wie
      - Eingaben mit Tastatur, Maus oder Gamepad
      - grafische Darstellung von Spielinhalten
      - Spielphysik
      - Spiellogik
      - Audio-Elementen
- Erstellung von Design Dokument

------------------------------------------------------------------------

## Features
- Rundenbasiertes Duell-System
- 5 Spiel-Modi
- Punktesystem
- AAA-Komponenten: Art, Animation, Audio
- UI/Text-Design
- Tutorial-Clip
- KI als Gegner
- Eingaben mit Maus & Tastatur
- Speichern von Einstellungswerten mit PlayerPrefs
